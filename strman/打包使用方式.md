## 目录结构

```ts
strMan-portable/
├── run.bat          # 启动脚本 (双击运行)
├── project/        # 源代码
│   ├── main.py
│   ├── config.yaml
│   └── src/
├── python/        # Python 3.11 + fastwhisper 模块
├── models/        # Whisper 模型 (~3GB)
└── cuda/         # CUDA 运行时
```

使用方法
## 双击 run.bat 或命令行
```ts
run.bat "C:\视频\video.mp4"
run.bat "video.mp4" --only-subtitle
run.bat "bilibili.com/video/BVxxx"
修改配置
编辑 project/config.yaml:
llm:
  openai:
    base_url: "http://192.168.1.11:9090/v1"  # 你的API地址
    api_key: "your-key"
    model: "你的模型"
注意

- 需要安装 FFmpeg 并添加到 PATH
- 需要 6GB+ 显存 GPU（使用 large-v3 模型）
- 纯 CPU 模式：将 whisper.device 改为 "cpu"
  文件总大小约 7GB，可以用 7-Zip 分卷压缩后分发。
```
